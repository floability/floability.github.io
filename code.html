To get started using Floability, follow the Quick Start README

  • floability-cli - Repository for the floability command-line tool for manipulating backpacks.
  • floability-examples - Repository of example backpacks for use with the floability-cli

Floability builds upon these other open source technologies:

  • depaul-dice/sciunit Sciunit repository from the DICE Lab at Mizzou.
  • cooperative-computing-lab/cctools TaskVine repository from the Cooperative Computing Lab at Notre Dame.
